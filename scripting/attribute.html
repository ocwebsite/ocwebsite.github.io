Attribute Table

We will pretend that "attribute" is an Attribute Table


attribute.name
The name of the attribute Attribute Names (string)

attribute.id
The id of the attribute Attribute IDs (number)

attribute.value
The value of the attribute (number)

attribute.default
The default of the attribute (number)

attribute.min
The minimum of the attribute (number)

attribute.max
The maximum of the attribute (number)
