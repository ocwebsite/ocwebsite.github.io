player Library

player.gamemode()
Returns the id of the gamemode the player is in (number)

player.position()
Returns the position of the player (ex: local x,y,z = player.position())

player.selectedPos()
Returns the position of the selected block of the player (ex: local x,y,z = player.selectedPos())

player.lookingPos()
Returns where the player raytrace hit (ex: local x,y,z = player.lookingPos())

player.rotation()
Returns the rotation of the player (ex: local yaw,pitch = player.rotation())

player.facingEntity()
Returns if the player is withing hit range of an entity (boolean)

player.facingBlock()
Returns if the player can start break a block (boolean)


player.name()
Returns the Xbox name of the player (string)

player.nametag()
Servers can chose this, will be empty for vanilla servers (string)


player.getFlag([number])
Returns the value of the flag, You can find a list of flags here (boolean)

player.effects()
Returns a table of Effects (table)

player.attributes()
Returns a table of Attribute Table (table)

player.inventory()
Returns an inventory table Inventory Table (table)
