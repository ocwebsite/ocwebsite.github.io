Globals

getClipboard()
Returns the user's clipboard (string)

setClipboard([string]text)
Sets the user's clipboard


importLib([string]name)
runs a file from the Scripts/Libs folder


tableToJson([table]table)
Converts a table to a json string

jsonToTable([string]json)
converts a json string into a table


sendLocalData([string]identifier, [string]message)
sends a message to other modules/command
the identifier can be anything and you check if it match the one it should on the receiver side
only string can be sent this way, be careful for infinite loops
