Color Setting

A color setting is different from normal colors

We will pretend that "color" is a Color Setting

The value of the color values range from 0.0 and 1.0
you can get this number from rgb by doing: color_value/255
(154 being an exemple value)


color.r = 1.0
The intensity of the red

color.g = 0.0
The intensity of the green

color.b = 0.0
The intensity of the blue

color.a = 1.0
The opacity/transparency


As this is a Color Setting, it has some extra fields

color.alphaVisible = true
Is true if the opacity slider should be hidden away from the user

color.rainbow = false
Is true when the color is rainbow (or RGB)

color.rainbowSpeed = 0.003
Speed at which rainbow (or RGB) alternates.
0.003 is the default value and 0 would probably crash or be static


Extras

You could create the value to assign like this:
color = {1, 0, 0, 0.5}
you could also add (in the same order the 3 extra fields)
like alphaVisible then rainbow then rainbowSpeed

Or just edit them by storing it, this allows you to change only
the values you want and left the others untouched
exemple changing to blue without touching other values :
local color = exemple.getColorSetting()
color.r = 0
color.g = 0
color.b = 1
