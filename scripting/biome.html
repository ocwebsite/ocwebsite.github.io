Biome Table

We will pretend that "biome" is a Biome

Note that this information is comming from the game's biome, not myself


biome.id
The runtime ID of the Biome (number)

biome.name
The name of the biome

biome.temperature
The temperature of that biome (number)

biome.snow
The snow accumulation of the biome (boolean)

biome.canRain
if rain is allowed in that biome (boolean)
