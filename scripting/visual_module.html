Visual Module

Everything in Module is in this too

We will pretend that "module" is a Visual Module


module.size
The size (as a table with a x and a y) of the module

module.pos
The position (as a table with a x and a y) of the module

module.relativepos
The relative position (as a table with a x and a y) from the anchor point (see below) of the module

module.anchor
The anchor of the module
Anchor Values:
0 Invalid
1 Top Left
2 Top Right
3 Bottom Left
4 Bottom Right
