File Table

We will pretend that "file" is a File


You can either call it like: file.close(file) or the easier way, file:close()
This exemple will use file: but just insert the file as first parameter if you use file.

You can get a file by using fs.open from the FileSystem library
Note that all files are in binary format
Make sure to read in the same order that you write and that the sizes are the same!


file:close()
Closes the file, important to close when done reading/writing, will flush for you

file:flush()
When you write it does not go to the hard drive instantly, this makes it go on it without closing

file:eof()
Will return true if it is the end of the file

file:size()
Will return the size of the file in bytes

file:seek()
Moves to a position in the file, 0 would be the begining

file:tell()
Tells you where you actually are in the file



Writing

file:write([string]Data, [number]amountOfBytes)
Writes data from a string in the file


You can chose Smaller formats if you dont need to store big numbers to save disk space (and read/write time)

file:writeByte([number]number)
Writes an 8 bit number (from -128 to 127)

file:writeShort([number]number)
Writes a 16 bit number (from -32768 to 32767)

file:writeInt([number]number)
Writes a 32 bit number (from -2147483648 to 2147483647)

file:writeFloat([number]number)
Writes a 32 bit floating point number (Has decimal points)

file:writeDouble([number]number)
Writes a 64 bit floating point number (Has decimal points)

file:writeString([number]number)
Writes a string (smaller then 65535 chars please)


Reading

Writes data from a string in the file


You can chose Smaller formats if you dont need to store big numbers to save disk space (and read/write time)

file:readByte([number]number)

file:readShort([number]number)
Reads a 16 bit number (from -32768 to 32767)

file:readInt([number]number)

file:readFloat([number]number)

file:readDouble([number]number)
